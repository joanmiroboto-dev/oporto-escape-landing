## Escapada a Oporto · 18–23 marzo 2026

Landing page turística moderna y responsive construida con **React**, **Vite**, **Tailwind CSS**, **React Router DOM** y **GSAP**.

Incluye:
- Ruta principal `/` con hero animado, explicación de por qué visitar Oporto, lugares imprescindibles, experiencia recomendada por días, galería visual y CTA final.
- Ruta `/formulario` con formulario de solicitud de información validado en frontend.

### Requisitos previos

- Node.js 18+ recomendado.

### Instalación

```bash
npm install
```

### Desarrollo

```bash
npm run dev
```

Después abre la URL que indique Vite (normalmente `http://localhost:5173`).

### Build de producción

```bash
npm run build
```
